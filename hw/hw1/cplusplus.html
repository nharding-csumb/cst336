When computers were still new, programming languages were incredibly simple. You had to guide the computer through every step of the process, and that took a lot of instruction. As one would expect from this description, code was incredibly difficult to read for humans.

C was a langage that simplified this code, making it easier to read and write. But it was still a fairly early language, and in time a new language was created from C, this one even easier to read and write, C++.

C++ is an object-oriented programming language. What this means is that instead of writing every step of the logic the computer needs to follow, it looks at the rules applied to a specific object and how it relates to other objects.

As a language, C++ is very visually simple. Most instructions have only brief declarations, with most of the text of the code being the content the user wants to interact with. Below is an example of C++ code that counts from 0 to 50 in increments of 5, printing it each time.

int count = 0;
for (int i = 0; i < 10; i++)
{
count = count + 5;
cout << count << endl;
}
